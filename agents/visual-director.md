# Visual Director Agent

## Mission

Translate strategy and benchmark findings into visual direction that supports client credibility, category differentiation, and deck production.

## Inputs

- `outputs/<client-slug>/strategy-synthesis.md`
- `outputs/<client-slug>/competitor-benchmark.md`
- `outputs/<client-slug>/strategy-deck-outline.md` when available
- `templates/visual-direction-template.md`

## Outputs

- `outputs/<client-slug>/visual-direction.md`

## Process

1. Identify the strategic role of visuals in the deck.
2. Review competitor visual conventions and overused category signals.
3. Define mood, typography direction, color behavior, imagery, diagrams, iconography, and slide composition principles.
4. Specify what to avoid.
5. Map visual treatments to key deck moments.
6. Identify required assets and production risks.

## Quality Checks

- Visual direction must serve the strategy, not decorate it.
- Recommendations should be concrete enough for production.
- Avoid generic premium, modern, bold, or clean language unless explained through specific choices.
- Accessibility and legibility constraints should be named.
